# Excel 导出

## 模块目标

实现任务列表 Excel 导出能力，导出数据必须复用任务列表筛选条件和权限范围，前端提供下载入口。

## 输入依据

- `plan.md`：建议加分功能包含 CSV 导出，任务接口包含 `GET /api/tasks/export`。
- `detailed-design.md`：详细设计将导出实现为 EasyExcel 导出 `.xlsx`。
- `detailed-design.md`：Excel 导出必须复用任务列表查询条件和权限过滤，避免导出越权数据。
- `detailed-design.md`：导出测试点包含文件响应头正确、导出数据遵守权限和筛选条件。

## 最小任务 checklist

- [x] 添加 EasyExcel 依赖。
- [x] 定义任务导出行 DTO 或 Excel 行模型。
- [x] 导出字段包含标题、描述、状态、优先级、负责人、创建人、截止日期、创建时间、更新时间。
- [x] 导出状态字段使用可读文本或明确枚举值。
- [x] 导出优先级字段使用可读文本或明确枚举值。
- [x] 导出逻辑复用任务列表筛选参数。
- [x] 导出逻辑复用当前用户权限范围。
- [x] 实现导师可按筛选条件导出全量可见任务。
- [x] 实现实习生只能导出分配给自己的任务。
- [x] 实现导出 Service。
- [x] 实现 Excel 文件写入 HTTP 响应。
- [x] 设置正确的 `Content-Type`。
- [x] 设置正确的 `Content-Disposition` 文件名。
- [x] 实现 `GET /api/tasks/export` 接口。
- [x] 导出接口支持 `status`、`assigneeId`、`dueDateStart`、`dueDateEnd`、`keyword` 参数。
- [ ] 前端封装导出 API。
- [ ] 前端任务列表页添加导出按钮。
- [ ] 前端导出时携带当前筛选条件。
- [ ] 前端处理二进制响应并触发文件下载。
- [ ] 前端导出失败时展示友好错误提示。
- [ ] 前端根据权限隐藏或禁用不允许导出的入口。

## 验收 checklist

- [ ] 点击前端导出按钮可以下载 `.xlsx` 文件。
- [x] 导出文件可以被 Excel 或兼容软件打开。
- [x] 导出数据字段完整。
- [x] 导出数据符合当前筛选条件。
- [x] 导师导出范围符合导师权限。
- [x] 实习生导出范围不会包含他人任务。
- [x] 响应头包含可识别的下载文件名。
- [ ] 导出失败时前端有明确提示。
